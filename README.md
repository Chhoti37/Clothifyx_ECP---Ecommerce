# 🛒 Clothifyx - Django E-Commerce Platform

**Clothifyx** is a feature-rich, full-stack E-commerce web application developed using the **Django** framework. 
It offers a smooth and secure shopping experience, from product discovery to successful checkout.

## 🚀 Key Features

* 👤 **User Authentication:** Secure Sign-up, Login, and Profile management system.
* 📦 **Product Catalog:** Dynamic product display categorized by Mobiles, Laptops, and Fashion.
* 🛒 **Smart Shopping Cart:** Seamlessly add, update, or remove items with real-time price calculation.
* 💳 **PayPal Integration:** Fully functional **PayPal Payment Gateway** integration (Sandbox mode) for secure transactions.
* 📝 **Order Management:** Automated order generation post-payment with a dedicated 'My Orders' history page.
* 📱 **Responsive UI:** Clean and modern interface built with **Bootstrap** to ensure a great experience on all devices.

## 🛠️ Tech Stack

Backend: I used Python and Django to handle all the server-side logic and database queries.

Frontend: The interface is built with HTML5 and CSS3 for a clean look.

Interactions: I integrated JavaScript for dynamic cart updates and the PayPal SDK for payments.

Design: Used Bootstrap 5 to make the website fully responsive and mobile-friendly.

Database: Used SQLite to manage products, user accounts, and order records.

## 💻 Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/Chhoti37/Clothifyx_ECP---Ecommerce.git](https://github.com/Chhoti37/Clothifyx_ECP---Ecommerce.git)
    ```

2.  **Create & Activate Virtual Environment:**
    ```bash
    python -m venv ecp_new
    # For Windows:
    ecp_new\Scripts\activate
    ```

3.  **Install Django:**
    ```bash
    pip install django
    ```

4.  **Run Migrations:**
    ```bash
    python manage.py migrate
    ```

5.  **Start the Server:**
    ```bash
    python manage.py runserver
    ```
